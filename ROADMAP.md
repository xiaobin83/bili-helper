## 演进路径

```
Skill 层（数据获取/操作）         ─→  Agent 层（智能分析/决策）
───────────────────────────────────────────────────────────────
视频评论员                        ─→  舆情自动化
牙尖帮                            ─→  社区文化雷达
```

## 原则

- 所有新项目优先复用 `bili-core/` 基础库，不自造轮子
- Skill 层聚焦数据获取与操作，Agent 层聚焦分析与决策
- 每个项目独立目录、独立 CLI，通过 OpenCode skill 系统注册
